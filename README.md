# Honeymoon the Band

Project repository for **Honeymoon the Band**.

Current site: https://www.honeymoontheband.com/

## Scope

This repo will house the band's web presence and supporting materials, including:

- A redesigned official website
- An EPK (electronic press kit)
- Other band-related projects as they come up

## Repository layout

```
assets/
  brand/                 Brand guidelines (PDF)
  logo/                  Logo files
  photos/
    band/                Group/band photos
    members/             Individual member headshots
```
